## 📋 Sobre o Projeto

Este projeto é uma **recriação fiel da landing page do Nubank**, desenvolvida com o objetivo de praticar habilidades de front-end como estruturação de HTML semântico, estilização com CSS e interatividade com JavaScript.

A página contém as seguintes seções:

- **Header** com menu de navegação responsivo (menu hambúrguer)
- **Hero** com chamada principal e formulário de CPF
- **Seção de produtos** (Conta, Cartão, Investimentos, Seguro, NuEmpresas, Shopping)
- **Seção do cartão de crédito** com card animado e lista de benefícios
- **CTA** para abertura de conta
- **Footer** completo com links e informações institucionais

## 🚀 Como rodar o projeto

Por ser um projeto estático (sem dependências ou build), basta abrir o arquivo diretamente no navegador:

**Opção 1 — Abrir direto:**
```bash
# Clone o repositório
git clone https://github.com/Almirodesouza/copyNubank.git

# Acesse a pasta
cd copyNubank

# Abra o arquivo index.html no seu navegador
```

**Opção 2 — Com Live Server (VS Code):**
1. Instale a extensão **Live Server** no VS Code
2. Clique com o botão direito no `index.html`
3. Selecione **"Open with Live Server"**
